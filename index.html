My Work

Reading Recs

Source Visit

For many busy people, a new book can feel like a risk. They don't want to waste their precious free time on a book that they hate. Reading Recs is a full-stack app which allows the user to suggest books, as well as see, like and comment on other user's recommendations. The project uses Node and Express to connect with the database on MongoDB on the backend and EJS on the frontend.

Source

This Google Chrome extension project uses a number of APIs, including APIs from The New York Times, Unsplash, and OpenWeather, to display information on the user's personal dashboard. Async JavaScript accesses and displays this information for the user.
